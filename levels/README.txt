Jelly No Puzzle level file format
---------------------------------
A level of H rows and W columns is described by H lines with W characters each,
terminated either by the end of the file or an empty line.

In the level description, characters have the following meaning:

 '#' is a wall
 '.' is an open space
 digits represent colored bricks:
   1 red
   2 green
   3 blue
   4 yellow
 (colors are immaterial for the solution but may be used to visualize solutions)
